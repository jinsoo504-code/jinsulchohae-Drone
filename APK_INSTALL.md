# 갤럭시 APK 설치 안내

## 다운로드

- APK 파일: https://github.com/jinsoo504-code/jinsulchohae-drone-app/releases/download/apk-20260610-005859/jinsulchohae-drone-app-debug.apk
- 릴리스 페이지: https://github.com/jinsoo504-code/jinsulchohae-drone-app/releases/tag/apk-20260610-005859
- 릴리스 노트: `APK_RELEASE_NOTES.md`
- 체크섬: `APK_CHECKSUMS.md`

## 설치 순서

1. 갤럭시에서 APK 다운로드 링크를 엽니다.
2. `jinsulchohae-drone-app-debug.apk`를 다운로드합니다.
3. 다운로드 완료 후 파일을 눌러 설치합니다.
4. 설치 차단 안내가 나오면 `설정`으로 이동합니다.
5. 사용 중인 브라우저 또는 파일 앱의 `이 출처 허용`을 켭니다.
6. 뒤로 돌아와 APK 설치를 다시 진행합니다.
7. 설치가 끝나면 앱을 열고 `샘플 데이터로 둘러보기`를 누릅니다.

## 갤럭시 점검 순서

1. 관리자 화면에서 `갤럭시 실기기 점검표 열기`를 누릅니다.
2. `권한/연결 상태 확인`을 눌러 위치 권한과 사진 권한을 확인합니다.
3. 대시보드, 지도, 작업 목록, 필지 상세, 사진 업로드, 방제팀 관리, 필지 등록을 순서대로 확인합니다.
4. 각 항목을 완료 체크하고, 문제가 있으면 현장 메모에 적습니다.
5. 마지막에 `점검 리포트 공유`를 눌러 결과를 공유합니다.

## 설치가 막힐 때

- `알 수 없는 앱 설치`가 막히면 갤럭시 설정에서 브라우저 또는 파일 앱의 설치 권한을 허용합니다.
- 사진 업로드가 안 되면 갤럭시 설정에서 앱의 사진 권한을 허용합니다.
- 위치 입력이 안 되면 갤럭시 설정에서 앱의 위치 권한을 허용합니다.
- APK 설치 대신 Expo Go로 확인하려면 `GALAXY_TEST.md`의 QR 실행 절차를 사용합니다.
