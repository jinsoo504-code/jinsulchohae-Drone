## 설치 순서

1. 갤럭시에서 APK 파일 링크를 엽니다.
2. APK를 다운로드합니다.
3. 다운로드된 APK 파일을 눌러 설치합니다.
4. 설치 차단 안내가 나오면 `설정`에서 브라우저 또는 파일 앱의 `이 출처 허용`을 켭니다.
5. 설치 후 앱을 열고 `샘플 데이터로 둘러보기`를 누릅니다.

## 확인할 현장 흐름

1. 대시보드에서 오늘 작업 현황을 확인합니다.
2. 지도에서 필지를 선택하고 상세 화면으로 이동합니다.
3. 작업 상태를 `방제 시작`, `방제 완료`, `문제 발생`으로 바꿔 봅니다.
4. 사진 업로드를 눌러 완료 사진 목록에 사진이 표시되는지 확인합니다.
5. 관리자 화면에서 방제팀 생성/수정 흐름을 확인합니다.
6. 필지 등록에서 농가, 필지, 작업 배정 흐름을 확인합니다.
7. `갤럭시 실기기 점검표`에서 항목별 완료와 현장 메모를 남깁니다.
8. 마지막에 `점검 리포트 공유`로 결과를 공유합니다.

## 문제 해결

- 사진 업로드가 안 되면 갤럭시 설정에서 앱의 사진 권한을 허용합니다.
- 위치 입력이 안 되면 갤럭시 설정에서 앱의 위치 권한을 허용합니다.
- APK 설치가 막히면 `알 수 없는 앱 설치` 권한을 허용합니다.
- APK 대신 Expo Go로 확인하려면 저장소의 `GALAXY_TEST.md`를 사용합니다.
