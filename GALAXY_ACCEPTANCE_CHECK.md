# 갤럭시 최종 인수점검

이 문서는 드론방제앱 MVP 목표 완료 여부를 갤럭시에서 확인하기 위한 최종 체크리스트입니다.

## 준비

1. `GALAXY_QUICK_INSTALL.md`에서 APK를 설치합니다.
2. 앱을 실행합니다.
3. 로그인 화면에서 `샘플 데이터로 둘러보기`를 누릅니다.
4. 관리자 화면에서 `갤럭시 실기기 점검표 열기`를 엽니다.
5. `권한/연결 상태 확인`을 눌러 위치 권한과 사진 권한을 확인합니다.

## 목표별 확인 항목

| 목표 항목 | 갤럭시 확인 방법 | 통과 기준 |
| --- | --- | --- |
| 로그인 | 로그인 화면에서 샘플 데이터 진입 | 대시보드로 이동 |
| 농가 등록 | 필지 등록 화면에서 새 농가명/전화번호 입력 | 저장 후 새 작업 생성 |
| 필지 등록 | 중심 좌표 입력 또는 현재 위치 사용 후 임시 필지 생성 | 지도/목록에 새 필지 반영 |
| 팀 등록 | 관리자 화면에서 방제팀 저장 | 팀 목록에 표시 |
| 작업 배정 | 필지 등록 시 담당 방제팀 선택 | 상세 화면 담당팀 표시 |
| 작업 목록 | 작업 목록 탭 확인 | 상태별/오늘 작업 확인 |
| 지도 | 지도 탭에서 필지 선택 | 하단 카드와 상세 이동 확인 |
| 상태 변경 | 필지 상세에서 방제 시작/완료/문제 발생 선택 | 대시보드/목록/상세 상태 반영 |
| 사진 업로드 | 필지 상세에서 사진 업로드 | 완료 사진 목록에 썸네일 표시 |
| 점검 기록 | 갤럭시 실기기 점검표 체크/메모 | 완료 시간과 메모 저장 |
| 리포트 공유 | 점검표에서 점검 리포트 공유 | 항목별 결과 공유 가능 |

## 완료 증거

최종 완료 처리를 하려면 아래 증거가 필요합니다.

- 갤럭시에서 앱 실행 성공
- 모든 목표 항목 통과
- 갤럭시 실기기 점검표 전체 항목 완료
- `점검 리포트 공유` 결과 보관

## 앱 내부 점검표 항목

관리자 화면의 `갤럭시 실기기 점검표`에는 아래 항목이 포함되어 있어야 합니다.

- 샘플 모드 진입
- 대시보드 확인
- 지도/필지 선택
- 작업 목록
- 상태 변경
- 사진 선택
- 방제팀 관리
- 필지 등록
- 작업 배정
- Supabase 연결 점검

## 실패 시 기록

실패한 항목이 있으면 갤럭시 실기기 점검표의 해당 항목 메모에 아래 내용을 적습니다.

- 어떤 화면에서 실패했는지
- 어떤 버튼을 눌렀는지
- 표시된 오류 문구
- 다시 시도했을 때 같은 문제가 반복되는지
