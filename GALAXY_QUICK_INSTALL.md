## 설치 후 확인

1. 다운로드된 `jinsulchohae-drone-app-debug.apk`를 눌러 설치합니다.
2. 설치 차단이 나오면 `설정`에서 `이 출처 허용`을 켭니다.
3. 앱을 열고 `샘플 데이터로 둘러보기`를 누릅니다.
4. 관리자 화면에서 `갤럭시 실기기 점검표 열기`를 누릅니다.
5. 권한/연결 상태 확인 후 모든 항목을 체크합니다.
6. 마지막에 `점검 리포트 공유`를 눌러 결과를 남깁니다.
